# 上海 2021 触发限售楼盘地图查询 Spec

## Why
用户需要快速了解“2021 年上海哪些新盘触发限售、在哪里、现在二手房挂牌价大概是多少”，并能用地图直观查看与筛选，方便后续自行部署与维护。

## What Changes
- 新建一个可部署到 Vercel 的 Web 项目（默认 Next.js + TypeScript）
- 新增“楼盘限售清单”数据导入与数据溯源字段（政府/官方房产平台链接）
- 新增地理编码（地址 -> 经纬度）与地图可视化（标记点 + 聚类）
- 新增二手房挂牌价数据接入层（可定期更新 + 手动刷新）
- 新增列表视图 / 地图视图切换、筛选（行政区/价格区间）、搜索
- 新增 CLI 命令用于查询/导出数据（便于脚本化与数据校验）

## Impact
- Affected specs: 数据采集与溯源、地理编码、价格更新、地图交互、筛选搜索、CLI、部署
- Affected code: 全新代码库（Next.js 前端 + Serverless API + 数据层），支持 Git 管理并可直接推送 GitHub

## ADDED Requirements

### Requirement: 项目形态与部署
系统 SHALL 以 Git 管理代码，并提供可在 Vercel 上构建与部署的应用形态（前端页面 + 后端 API）。

#### Scenario: Vercel 部署成功
- **WHEN** 在 Vercel 绑定 GitHub 仓库并配置必要环境变量
- **THEN** 应用可访问首页并加载地图与列表数据

### Requirement: 限售楼盘数据模型（含溯源）
系统 SHALL 存储并展示每个限售楼盘的基础信息，且每条记录包含可追溯的数据来源（政府公示网站或官方房产平台链接）。

数据字段（最小集合）：
- 楼盘名称（name）
- 限售触发时间（triggeredAt，精确到日；无法精确则记录到月并标注精度）
- 行政区（district）
- 地址（address）
- 经纬度（lng/lat）
- 数据来源（sourceTitle/sourceUrl/sourcePublishedAt，可选）
- 楼盘照片（photoUrl，可选）

#### Scenario: 读取楼盘详情
- **WHEN** 用户查看某个楼盘详情
- **THEN** 能看到以上字段及其来源链接

### Requirement: 坐标获取（地理编码）
系统 SHALL 为每条楼盘记录提供经纬度，用于地图标记；若原始数据仅含地址，系统 SHALL 通过地图服务提供的地理编码能力补全坐标。

#### Scenario: 导入数据后补全坐标
- **WHEN** 导入的楼盘缺少经纬度但有地址
- **THEN** 系统补全经纬度并持久化

### Requirement: 二手房挂牌价数据接入与更新
系统 SHALL 为每个楼盘提供“当前二手房挂牌价”字段，并支持定期自动更新与手动刷新。

说明（约束）：
- “挂牌价”数据来源 SHALL 记录溯源字段（priceSourceTitle/priceSourceUrl/priceUpdatedAt）。
- 若无法从政府/官方房产平台获得可公开访问的“按楼盘维度挂牌价”，系统 SHALL 支持以“官方房产平台/权威平台公开页面”作为数据源，并在 UI 中明确标注“价格来源与更新时间”。（不在系统中存储或展示任何需登录/付费/绕过反爬获取的数据）

#### Scenario: 定期更新价格
- **WHEN** 触发定时任务
- **THEN** 系统更新价格并记录 priceUpdatedAt

#### Scenario: 手动刷新价格
- **WHEN** 用户点击“刷新”
- **THEN** 系统重新拉取并更新价格，同时展示加载状态

### Requirement: 地图展示与交互
系统 SHALL 集成高德/百度/Google 地图之一的 Web API（默认高德地图，因数据在中国大陆可用性更好）。

地图能力：
- 支持缩放、拖动
- 用标记点展示全部限售楼盘
- 标记点聚类（cluster）优化
- 点击标记点展示详情卡片：楼盘名称、限售触发时间、挂牌价、行政区、照片（如有）、来源链接

#### Scenario: 点击标记点查看详情
- **WHEN** 用户点击任一标记点
- **THEN** 弹出详情卡片且信息完整、来源可点击

### Requirement: 列表视图与筛选搜索
系统 SHALL 提供列表视图与地图视图切换，并支持以下交互：
- 按行政区筛选
- 按价格区间筛选（支持“未知价格”单独开关）
- 关键字搜索（楼盘名称/别名）
- 分页或懒加载（避免一次性渲染过多 DOM）

#### Scenario: 通过搜索定位楼盘
- **WHEN** 用户输入楼盘名称关键字
- **THEN** 列表与地图仅展示匹配项（地图视图可自动聚焦到匹配结果的范围）

### Requirement: 最后更新时间展示
系统 SHALL 在界面上展示“数据最后更新时间”（至少包括：楼盘清单更新时间、价格更新时间）。

#### Scenario: 用户查看更新时间
- **WHEN** 用户打开页面
- **THEN** 在显著位置看到更新时间信息

### Requirement: CLI 命令查询与导出
系统 SHALL 提供命令行入口，支持对数据进行查询与导出，以便离线核对与自动化处理。

最小命令集合：
- 按年份列出触发限售的楼盘（默认 2021）
- 支持按行政区、价格区间、关键字过滤
- 支持导出 JSON/CSV

#### Scenario: CLI 查询 2021 限售楼盘
- **WHEN** 运行命令并指定 year=2021
- **THEN** 输出楼盘清单（含经纬度与挂牌价字段）
